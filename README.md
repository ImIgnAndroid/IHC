# Estructura del proyecto IHC

Carpetas principales:

- `public/`: vistas HTML.
- `assets/css/`: estilos compartidos.
- `assets/js/`: scripts compartidos.
- `assets/img/`: imágenes.
- `docs/`: documentación e historias de usuario.
- `data/`: datos de traducciones y catálogo.

Notas:
- Las páginas HTML en `public/` usan rutas relativas a `../assets/`.
- Mantén el contenido de docs/ separado de los archivos de código.
